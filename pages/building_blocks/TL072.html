TL072 AMPLIFIER

TL072 AMPLIFIER

This circuit is designed as a simple way to make any signal louder, which can be very useful in a lot of different applications like mixers and effects pedals. Boosting the signal before an audio circuit can help it work better with the quieter instrument level signals produced by guitars, and boosting it afterwards can fix circuits which are too quiet to use reliably with other equipment.

DETAILS

Operational amplifiers are IC chips which take an input signal and spit out the same signal only louder. The TL072 is a very common opamp found in in synthesisers, guitar pedals and other audio circuits; but all opamaps do pretty much the same thing so this should work with other 8 pin chips like the LM358 or NE5532. Other chips have different sounds and characteristics so it's worth experimenting with them, just check the datasheets before you do to make sure that the pins and supply voltages match up.

The TL072 is a dual opamp which means that it contains two identical opamp circuits, each represented by the triangles on the schematic. The second of these is the part which actually makes the signal louder, and the first is used as a buffer to isolate the amplifier and match the impedances of whatever it's connected to.

Impedance is a measure of how much a circuit will oppose a current. It's very similar to normal resistance, only it also accounts for AC circuit components like capacitors and inductors. The important part is that if the impedance of two circuits is different then it can cause a lot of problems when you connect them together. If you've ever built something that sounds too quiet and a bit broken or scratchy, it might be because the impedances were not correctly matched throughout the circuit. The buffer is an amplifier with no gain (it doesn't make the signal any louder or quieter) which will match the impedance of the circuits it is placed between so they can work together without problems. Buffers are particularly important when designing circuits to work with electric guitars as the pickups of the guitar have a large impedance, which can cause problems when connected to low impedance audio circuits.

The second part of the circuit amplifies the signal. Varying the resistance between pins 1 and 2 (the output and inverting input) changes the gain of the amplifier, making the output louder or quieter. With zero resistance the gain is 0 (which is how the buffer circuit is set up), and as the resistance is increased the gain is also increased. Although the gain can be theoretically infinite if the two pins are disconnected from each other, after a certain point the top and bottoms of the waves will be 'clipped' off which will distort the sound. The 25k potentiometer used in this circuit has a high enough resistance to push the chip into distortion because I quite like pushing things into distortion, but if you want a cleaner (although quieter) amplifier you can use something smaller like a 10k or 15k potentiometer instead.
